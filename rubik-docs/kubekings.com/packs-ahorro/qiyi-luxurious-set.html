QiYi Luxurious Set

Con el Pack QiYi Luxurious Set conseguirás algunos de los más populares cubos de rubik de QiYi que no pueden faltar en ninguna colección. Este pack contiene 8 cubos de rubik básicos como pyraminx, megaminx, skewb y otros. Cada cubo tiene su caja individual por lo que podrás separar este pack para regalar parcialmente si así lo deseas. Este pack contiene:

- QiYi QiHeng Megaminx

- QiYi QiMing Pyraminx

- QiYi QiCheng Skewb

- QiYi Mirror 3x3

- QiYi Ivy Cube

- QiYi QiDi 2x2

- QiYi Sail 3x3

- QiYi QiYuan 4x4
