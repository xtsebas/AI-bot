Descripción LeFun Time Machine 2x2

LeFun Time Machine 2x2 es un cubo muy bonito que cualquier amante de los cubos de rubik querrá tener en su colección. La versión original de este rompecabezas fue de smaz y está fuera de producción desde hace mucho tiempo, ahora gracias a Lefun podemos disfrutar de esta versión stickerless que es mucho más barata y gira genial.

En kubekings hemos diseñado unas pegatinas para la versión stickerless, de esta forma puedes darle una solución un poco más complicada ya que las ruedas irán numeradas y las piezas tendrán posición.

Las sticker Time Machine 2x2 son muy bonitas y le dan un aspecto genial, podrás poner el negativo o el positivo de la pegatinas, haciendo que tenga sticker out-line o no.
