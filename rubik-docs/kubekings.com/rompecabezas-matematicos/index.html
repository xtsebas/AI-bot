Rompecabezas de colocación diseñado por Jürgen Reiche. Este juego consta de 16 fichas grises y 16 rojas, con 8 fichas de cada color numeradas del 1 al 4 en una cara. Has resuelto la tarea cuando el tablero está completamente cubierto y, para cada número descubierto, en la fila horizontal y vertical de cada número debe haber tantas fichas del color opuesto como el número indicado en la ficha.
